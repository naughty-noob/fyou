## How to run

**Python:**
```bash
cd Python && python main.py
```

**JavaScript:**
```bash
cd "JavaScript " && node main.js
```

**C:**
```bash
cd C && gcc main.c -o main && ./main
```

**Bash:**
```bash
cd Bash && chmod +x rude.sh && ./rude.sh
```
